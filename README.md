# ProductSuite — PSDev Sistemas

Plataforma de geração de imagens e vídeos com IA por workflow visual, desenvolvida pela **PSDev Sistemas**.

## Tecnologias

- React 18 + TypeScript + Vite
- React Flow (workflow visual)
- Google Gemini API (geração de imagens e vídeos)
- Firebase Authentication
- TailwindCSS + shadcn/ui

## Desenvolvimento

```bash
npm install
npm run dev
```

Configure as variáveis de ambiente em `.env`:

```
VITE_GEMINI_API_KEY=sua_chave_aqui
VITE_FIREBASE_API_KEY=...
```
